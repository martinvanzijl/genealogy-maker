Genealogy Maker – Manual

Overview

Genealogy Maker is a program for creating genealogies.

Adding a Person

To add a person to the genealogy:

  1. 1.Click on “Add Item” on the left toolbox. 

  2. 2.Click on the “Person” button. 

  3. 3.Click on the diagram. The person will be added to the diagram where you clicked. 

  4. 4.Type the person’s name. 

  5. 5.Press Enter. The person’s name will be set. 

Renaming a Person

To change a person’s name:

  1. 1.Double-click on the person in the diagram. This should show the details window. 

  2. 2.Enter the new name in the “name” box. 

  3. 3.Click “Save”. 

Editing a Person’s Details

To edit the details of a person:

  1. 1.Right-click on the person in the diagram. 

  2. 2.Select “Details...”. This should show the details window. There are three tabs: Details, Biography and Photos. 

  3. 3.In the Details tab, you can enter the name, gender, date and place of birth, and date and place of death of the person. 

  4. 4.In the Biography tab, you can enter a description of the person. 

  5. 5.In the Photos tab, you can attach photos of the person. To do this, click the “Add...” button and choose the photo to add. 

  6. 6.To remove a photo, select the photo in the list and click “Remove”. 

  7. 7.To view a photo, double-click on it in the list. This shows the photo viewer window. See the “Photo Viewer” section below for more details. 

  8. 8.Click “Save” or press Ctrl+S. 

About the Name Entries

There are three name entries for each person:

  1. 1.First name 

  2. 2.Last name 

  3. 3.Display name 

The first name and last name are used when exporting to GEDCOM files.

The display name is shown on the diagram.

About the Photo Viewer Window

The photo viewer window shows the photo you double-clicked on in the Person Details window. This has the following menu options:

  • •Use “File → Save As…” to save a copy of the photo. 

  • •Use “File → Close” to close the window. 

  • •Use “Edit → Copy” to copy the photo to the clipboard. 

  • •Use “View → Zoom In” and “View → Zoom Out” to zoom in and out. 

  • •Use “View → Normal Size” to reset the zoom. 

  • •Use “View → Fit to Window” to make the photo fit exactly in the window. Click this option again to undo it. 

  • •Use “Navigate → Go to Next” to go to the next photo in the list (if one exists). 

  • •Use “Navigate → Go to Previous” to go to the previous photo in the list (if one exists). 

  • •Use “Navigate → Show Navigation Panel” to show a panel with “Previous” and “Next” buttons at the bottom of the window. Clicking “Next” goes to the next photo, and “Previous” goes to the previous photo. Click the menu item again to hide the panel. 

Changing the Fill Color of a Person

You can change the fill color of a person’s box by clicking on the “Fill Color” icon in the toolbar (shown below), then choosing a color from the fill color window and clicking “Select”.

By default, the fill color is white.

 

Removing a Person

To remove a person:

  1. 1.Right-click on the person in the diagram. 

  2. 2.Select “Delete”. 

Or:

  1. 1.Click on the person in the diagram. 

  2. 2.From the menu bar, select “Item → Delete”. 

Or:

  1. 1.Click on the person in the diagram. 

  2. 2.Press the <Delete> key. 

Adding a Parent/Child Relationship

A relationship goes from parent to child. To add a relationship:

  1. 1.Click on “Add Item” on the left toolbox. 

  2. 2.Click the “Parent/Child Relationship” button. 

  3. 3.Click on the parent in the diagram, then drag the mouse to the child. This will add an arrow from the parent to the child. 

Removing a Relationship

To remove a relationship:

  1. 1.Click on the relationship. 

  2. 2.Click the “Delete” button, or press the <Delete> key, or from the menu bar select “Item → Delete”. 

Adding a Marriage

To show that two people are married, do the following:

  1. 1.Click and drag one spouse onto the other spouse. The other spouse should be highlighted. 

  2. 2.Release the mouse button. This should show a dialog to ask whether to marry the two people or not. 

  3. 3.Click “Yes”. The spouses should appear next to each other, and there should be a “wedding ring” between them. 

When you move one spouse, the other spouse will automatically move along with them.

Removing a Marriage

To remove a marriage between two people:

  1. 1.Right-click on the circle (“wedding ring”) between the two spouses. 

  2. 2.Select “Break Marriage”. The circle should be removed. 

Editing a Marriage

To edit the details of a marriage:

  1. 1.Right-click on the circle (“wedding ring”). 

  2. 2.Select “Details...”. This should show the “Marriage Details” form. 

  3. 3.Enter the date and place of the marriage in the form. 

  4. 4.Click “Save”. 

Undo an Action

You can undo most things, such as adding a person, deleting items or moving items. To undo a step:

  1. 1.From the menu bar, select “Edit → Undo”. The menu item will show a description of the last step, such as “Undo add person”. 

To redo the action again:

  1. 1.Select “Edit → Redo”.The menu item will show a description of the last step, such as “Redo add person”. 

Selecting Items

To select a single person or relationship, click on the person or relationship in the diagram.

To select multiple items, hold <Ctrl> while clicking on the items. You can also click and drag with the left mouse button to select all items in a rectangular area.

To select all items in the diagram, from the menu bar select “Select → Select All”.

Do deselect all items, from the menu bar select “Select → Select None”.

Changing the Canvas Size

You can change the canvas size (the limits of the diagram) as per below:

  1. 1.From the menu bar, select “Diagram → Change Size...” 

  2. 2.Enter the new width and height. 

  3. 3.Click “OK”. 

Finding a Person

To find a person with a given name:

  1. 1.From the menu bar, select “Edit → Find...”, or press <Ctrl+F>. 

  2. 2.Enter the person’s name. 

  3. 3.Click “Find”. This should show the person in the diagram. 

  4. 4.Click “Close” to close the box. 

You can also locate the person by doing the following:

  1. 1.Click on the “Tree View” tab on the left. This will show a list of people in the diagram. 

  2. 2.Double-click on the person in the tree view. This will navigate to that person in the diagram and briefly highlight the person with a colored border. 

Making the Find Window Transparent

Sometimes the found person will be behind the “Find” window. To make the window transparent while the person is highlighted:

  1. 1.Select “Edit → Preferences...” from the menu. 

  2. 2.Check the box “Make ‘Find’ window partly transparent. 

  3. 3.Click “Apply”. 

  4. 4.Click “Close”. 

Changing the Background

To change the background of the diagram:

  1. 1.Click on the “Backgrounds” tab on the left. 

  2. 2.Click on the desired background. This should change the background on the diagram. 

Aligning Items

To align items:

  1. 1.Select the items in the diagram. 

  2. 2.From the menu bar, select “Align” → “Align Horizontally” or “Align → Align Vertically”. 

Zooming

To zoom on the diagram, either:

  1. 1.Select a zoom level from the combo-box on the top right. 

Or:

  1. 2.Type in a zoom level to the combo-box on the top right. 

Or:

  1. 3.Press <Ctrl> while scrolling up or down with the mouse wheel. 

Or:

  1. 4.Click and drag the slider on the top right. 

Panning

To pan from side to side or up and down in the diagram, either:

  1. 1.Use the scroll-bars at the bottom and right, or: 

  2. 2.Scroll the middle mouse button up and down, or: 

  3. 3.Click and drag the middle mouse button. 

Creating a New Genealogy

To create a new genealogy, select “File → New” from the menu, or press <Ctrl+N>.

Opening an Existing Genealogy

To open an existing genealogy, select “File → Open” from the menu, or press <Ctrl+O>. Choose the file you want to open.

You can also open a recently used diagram by selecting “File → Recent Files” and choosing the diagram from the menu.

Saving a Genealogy

To save a genealogy, select “File → Save” from the menu, or press <Ctrl+S>. The genealogy is saved as an XML file.

Use “File → Save As...” or <Ctrl+Shift+S> to save the genealogy as a different file.

Changing Arrow Line Thickness

To change the thickness of the arrows (relationship lines):

  1. 1.Select “Edit → Preferences” from the menu. 

  2. 2.Enter the new line thickness. 

  3. 3.Click “Apply”. This will change the line thickness of all arrows. 

  4. 4.Click “Close”. 

Importing GEDCOM Files

To import a GEDCOM file:

  1. 1.Select “File → Import GEDCOM File...” from the menu. 

  2. 2.Choose the file to import. 

  3. 3.Click “Open”. The GEDCOM file should be imported and the diagram shown. 

NOTE: Only the latest marriage for any given person is imported. Earlier marriages are ignored. This means that the imported file may not completely reflect the GEDCOM file.

NOTE: The “Country of Birth” field is not populated when importing GEDCOM files.

Exporting GEDCOM Files

To export a genealogy as a GEDCOM file:

  1. 1.Select “File → Export GEDCOM File...” from the menu. 

  2. 2.Choose the file name to export to. 

  3. 3.Click “Save”. The GEDCOM file should be exported. 

Showing Thumbnails in Diagram

The diagram will show a thumbnail of each person on the left of their box in the diagram, if the person has photos attached. The first photo in the list will be used as the thumbnail image.

To disable this:

  1. 1.Select “Edit → Preferences...” from the menu. 

  2. 2.Un-check the box “Show thumbnail photos in diagram”. 

  3. 3.Click “Apply”. 

  4. 4.Click “Close”. 

Sharing a Genealogy File

To copy a genealogy file from one PC to another:

  1. 1.Find the XML file where you saved the genealogy.  

    If you have added photos to the genealogy, these will be saved in a folder alongside the XML file. For example, if the XML file is “family-tree.xml”, the photos will be stored in the folder “family-tree-photos”. 

  2. 2.Copy both the XML file and the photos folder (if it exists). Put the copy on the other PC. 

  3. 3.Open the copied XML file with Genealogy Maker on the other PC. 

Hiding the Sidebar

You can hide the sidebar at the left to give more space to the diagram. To do this:

  1. 1.Select “Edit → Preferences” from the menu. 

  2. 2.Check the box “Show sidebar collapse/expand button”. 

  3. 3.Click “Apply” 

  4. 4.Click “Close”. 

  5. 5.Click the button labeled “<<” next to the sidebar. This will collapse the sidebar. 

  6. 6.Click the button again to show the sidebar again. 

You can also choose “View → Show Sidebar” from the top menu to show or hide the sidebar.

Sorting the Tree View

You can click the “Tree View” tab on the left to show a list of people in the diagram.

This is sorted by alphabetical order. You can click on the header to reverse the sort order.

Exporting Images

To export the diagram as a PNG image:

  1. 1.Select “Diagram → Export Image...” from the menu. 

  2. 2.Choose the destination file name. 

  3. 3.Click “Save”. 

A message box should show to say that the image has been exported.

NOTE: If the diagram is empty, nothing will be exported.

NOTE: The image will be just large enough to contain the diagram contents, without any padding on the sides or the top or bottom.

NOTE: You can also choose “File → Export Image...” from the menu.

Creating Reports

To create a report to show the people in the diagram:

  1. 1.Select “Reports → Person List” from the menu. 

This will show the report in a new window. Click the “Close” button at the bottom to close the report.

You can click on the table headers to change the sort order.

NOTE: Default birth and death dates will be shown as empty in the report.

To create a report showing the timeline of events (births, deaths and marriages):

  1. 1.Select “Reports → Timeline” from the menu. 

This will also show the report in a new window.

Viewing File Properties

To view the properties of the current diagram file:

  1. 1.Select “File → File Properties...” from the menu. 

This will show a window with the name, folder and timestamp of the current diagram. If the diagram has not been saved yet, these will be empty.

This also shows the number of people, parent/child relationships and marriages in the current diagram.

Running from the Command Line

When running from the command line or shell, you can specify the starting file to open. For example:

./genealogymaker my-family-diagram.xml

This will load “my-family-diagram.xml”. The program will only load the file if it exists and is a valid genealogy file.

Example Diagrams

You can open an example genealogy diagram from the “examples” folder.

  1. 1.Select “Help → Open Example...” from the menu. 

  2. 2.Choose an example file from the list. 

  3. 3.Click “Open”. 

Automatically Laying out Diagrams

You can automatically lay out a diagram by selecting “Diagram → Auto-layout” from the menu. This will arrange the persons from left to right, and place parents above their children. (NOTE: People without descendants will be put on the bottom row.)

To undo this, select “Edit → Undo” from the menu.

Removing Items from the “Recent Files” Menu

To clear the “Recent Files” menu, open the menu and select “Clear”, then click “Yes”.

To remove non-existing files from the menu, select “Edit → Preferences...”, then check the box “Remove invalid files from 'Recent Files' menu”. Now, if you select a recent file which does not exist, it will be removed from the menu.

Selecting Descendants

To select the descendants of a person:

  1. 1.Right-click on the person in the diagram. 

  2. 2.Choose “Select Descendants”. 

This will select all descendants of the person. If the person is married, it will also select the descendants of their spouse.

You can then, for example, click the “line color” button in the toolbar and select a color, to give the family tree a specific color.

Setting the Diagram Font

To change the font used on the diagram (e.g. for person names):

  1. 1.From the menu, select “Edit → Preferences”. 

  2. 2.In the section named “Diagram font”, choose the font and size you would like. 

  3. 3.Click “Apply”.
